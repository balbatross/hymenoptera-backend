## Hymenoptera Flow Backend

Backend server for Hymenoptera.

### Contains 

- Flow packager
- Flow engine
- Base modules
- Flow sanbox

### Usage

`npm install`

`node src/http.js`

You may need to run the above command twice until the local npm repository is changed
